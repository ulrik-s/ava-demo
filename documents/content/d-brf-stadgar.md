# BRF Vinkeln — Stadgar

Antagna vid årsstämman 2018.

## §32 Underhållsskyldighet

Föreningen ansvarar för underhåll av:
- Stamledningar (vatten, värme, avlopp)
- Yttertak och fasader
- Trapphus och gemensamma utrymmen

Bostadsrättshavaren ansvarar för underhåll inne i lägenheten, dock ej för stamledningarnas funktion.
